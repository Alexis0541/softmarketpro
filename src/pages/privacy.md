_Ultima actualizacion_: 14 de julio de 2026

SoftMarket Pro recopila los datos necesarios para gestionar pedidos, enviar licencias digitales y ofrecer soporte al cliente.

## Datos que puede solicitar la tienda

- Nombre y apellidos.
- Correo electronico.
- Telefono.
- Pais y direccion de facturacion.
- Productos comprados y datos del pedido.

## Uso de los datos

Los datos se usan para procesar compras, enviar licencias, emitir facturas, responder consultas y mejorar el servicio.

## Conservacion

Los datos de pedidos y facturacion pueden conservarse durante el tiempo necesario para cumplir obligaciones legales y comerciales.

## Derechos del usuario

El cliente puede solicitar acceso, rectificacion o eliminacion de sus datos mediante la pagina de contacto.

## Seguridad

La tienda debe conectarse a una pasarela de pago segura antes de procesar pagos reales. No se deben guardar datos completos de tarjetas en el sitio.
